Genre

Physics-Based Slingshot... game

Everyone that has heard me explain it goes straight to comparing it to Angry Birds, so that's how I'm going to start off. It's like Angry Birds in the sense that you are slingshotting an object, but in no other ways is it really "like angry birds". You will being attempting to get an object to reach specific destinations depending on the point value each destination holds. Watch out though, you could lose points as well!

Platform

Desktop Browser

Currently, it is only planned to be able to be played within a desktop browser mouse input (click and drag)

Esthetics

Simple sprite based Art and 8-bit style audio tracks
